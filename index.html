Frontend Development

Frontend development is the development of the Graphical user interface of a website, through hte use of HTML,CSS,and Java Script, so that users can view and interact with the website.

  • HTML: The HyperText Markup Language or HTML is the standard markeup language forn documents designed to be displayed in a web browser.

  • CSS: Cascading Style Sheets (CSS)is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML

  1. Pure CSS

  2. Bootstrap (Framework)

  3. Tailwind (Framework)

  4. Sass (Preprocessor)

  • Java Script : Java Script often abbreviated JS, is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. it is used to add functionality in the website.
  1. Vanilla JS

  2. jQuery (Library)

  3. React (Framework)

  4. Angular (Framework)

  5. Vue (Framework)

  6. Typecript (Preprocessor)
